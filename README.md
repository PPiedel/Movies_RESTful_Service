# Movies RESTful Service
RESTful web service following HATEOAS principles.
# Functionalities
- get all movies
- get movie details
- post movie
# Technologies : 
- Java 9
- Spring Boot 2
- Hibernate 5 + JPA
- JUnit 4
- Maven

# Starting service
To run service you can just execute following command in project directory.
```bash
./start.sh
